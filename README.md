# ⚙️ Feature Engineering Projects

This repository contains a collection of Feature Engineering projects focused on transforming raw data into meaningful features for machine learning models.

The objective is to improve model performance by applying effective preprocessing, encoding, and feature transformation techniques.

---

## 🚀 Objectives

- Clean and preprocess raw datasets
- Create new meaningful features
- Handle categorical and numerical variables
- Apply scaling and normalization techniques
- Prepare data for machine learning models

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Feature-engine (optional)
- Jupyter Notebook

---

---

## 🔧 Key Techniques Covered

### 📌 Data Cleaning
- Handling missing values (mean, median, mode, interpolation)
- Removing duplicates
- Outlier detection and treatment

### 🔢 Numerical Feature Engineering
- Scaling (Standardization, Min-Max Scaling)
- Log transformations
- Binning and discretization

### 🔤 Categorical Feature Engineering
- Label Encoding
- One-Hot Encoding
- Target Encoding
- Frequency Encoding

### 🧠 Feature Creation
- Polynomial features
- Interaction features
- Date-time feature extraction
- Domain-specific feature generation

### ⚡ Feature Selection
- Correlation-based selection
- Recursive Feature Elimination (RFE)
- Feature importance (tree-based models)

---

## 📊 Workflow

1. Data Understanding
2. Data Cleaning
3. Feature Creation
4. Feature Transformation
5. Feature Selection
6. Final Dataset Preparation

## 💡 Key Learnings

- Importance of feature quality over quantity

- Handling real-world messy datasets
